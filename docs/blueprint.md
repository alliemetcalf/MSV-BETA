# **App Name**: LoginFlow

## Core Features:

- User Authentication: Allows users to log in securely with their credentials using Firebase authentication.
- Success Message: Displays a success message upon successful login to confirm user authentication.
- Logout Functionality: Provides a logout button for users to securely sign out of their account.
- Firebase Integration: Connects to the specified Firebase project (project number 614074080882, project name msv-beta) for authentication services.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a trustworthy and secure feel.
- Background color: Light blue (#E8EAF6), a desaturated tint of the primary color, providing a calm and clean backdrop.
- Accent color: Purple (#7E57C2), an analogous hue to the primary color, used for interactive elements like the logout button.
- Body and headline font: 'Inter' for a modern, neutral, and readable interface. 
- Use simple, clear icons for login status and logout actions.
- Clean and straightforward layout with centered login form and success message.
- Subtle animations for form transitions and button interactions.
